Outils utilisés

Revit, AutoCAD, SketchUp, QGIS, Blender, Visual Studio Code

Sur ce site, vous trouverez :

À gauche, un panneau avec des onglets contenant des scènes 3D. J'ai opté pour un rendu low-poly avec Three.js pour une exécution rapide et fluide.

Chaque maquette est interactive : vous pouvez zoomer, la faire pivoter, etc.

Consultez la section Aide pour découvrir toutes les commandes disponibles.

À droite, vous trouverez la section 2D, composée d'onglets dévoilant différents plans de différents types.

Pour la 2D, j'ai choisi la technologie SVG, qui permet un niveau de zoom illimité et garantit une qualité d'image constante.

J'ai conçu ce site avec Visual Studio Code, ce qui m'a permis d'apprendre et d'utiliser HTML5, CSS et JavaScript.

Bien que je ne maîtrise pas encore parfaitement ces langages, j'ai acquis les compétences nécessaires pour les mettre en œuvre dans ce projet.

Je vous invite à explorer le site et à profiter de cette expérience visuelle. J'espère que votre visite vous plaira !
